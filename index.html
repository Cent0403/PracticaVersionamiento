Materias Aprobadas

Base de Datos

Materia enfocada en la organización, modelado y gestión de la información mediante sistemas de gestión de bases de datos.

Lógica de la Programación

Fundamentos de pensamiento lógico para resolver problemas con algoritmos y estructuras secuenciales, condicionales y repetitivas.

Programación Orientada a Objetos

Paradigma de programación que organiza el código en clases y objetos, aplicando conceptos como herencia, encapsulamiento y polimorfismo.

Programación Orientada a Eventos

Modelo de programación donde las acciones dependen de eventos externos como clics, entradas de usuario o mensajes del sistema.

Diseño Web

Materia centrada en la creación de páginas web atractivas y funcionales utilizando HTML, CSS y principios de usabilidad.
